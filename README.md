# student-planner
A full-stack web application for managing student tasks, deadlines and academic progress.

Project Overview

The Student Planner allows users to create, update, organise and track their academic tasks. Students can assign subjects, deadlines and priority levels to tasks and monitor their overall progress.

Planned Features
1. User registration and login
2. Add, edit and delete tasks
3. Assign tasks to subjects
4. Set task deadlines
5. Set low, medium or high priority
6. Mark tasks as completed
7. View upcoming and overdue tasks
8. Filter tasks by subject, status and priority
9. Search for tasks
10. View task completion statistics

Technologies
